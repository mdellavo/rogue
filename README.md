# Rogue

a rogue-like/lite with an asyncio python backend, javscript/websocket based frontend.

## Features

- multiplayer, async backend
- procedurally generated maps
- realtime websocket base gameplay

## Development

```bash
# run backend
python -m rogue

# run react dev server
yarn start
```
